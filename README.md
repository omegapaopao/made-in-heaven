# made in heaven

> 网页视频倍速控制脚本 — 自动识别视频，精致面板，快捷键操控。

by **omegapaopao**

---

## 功能

- 自动检测网页中的视频元素
- 倍速播放：0.5× / 0.75× / 1× / 1.25× / 1.5× / 2× / 3× / 5×
- 逐级微调倍速（±0.25）
- 自定义倍速输入（任意 0.1~16× 数值）
- 播放/暂停、快进/快退
- 进度条点击跳转
- 画中画、全屏、静音
- 全键盘快捷键操控
- 毛玻璃渐变 UI，可拖拽、折叠、关闭

---

## 快捷键

| 按键 | 功能 |
|------|------|
| `空格` | 播放 / 暂停 |
| `←` `→` | 快退 / 快进 5 秒 |
| `Ctrl` + `←` `→` | 快退 / 快进 15 秒 |
| `↑` `↓` | 倍速 +0.25 / -0.25 |
| `1` `2` `3` `5` | 快速切换倍速 1× 2× 3× 5× |
| `R` | 重置 1 倍速 |
| `F` | 全屏 |
| `P` | 画中画 |
| `M` | 静音 |

## 界面

- 紫色渐变 Logo + 毛玻璃面板
- 大字号倍速显示，当前预设高亮发光
- 可拖拽标题栏移动位置
- 自定义倍速输入框，输入任意数值后回车
- 点击 `✕` 关闭面板，显示 ⚡ 小圆点
- 点击小圆点恢复面板

---

## 适用网站

Bilibili · YouTube · 抖音 · 西瓜视频 · 微博 · Twitter/X · Instagram · 各类影视网站
